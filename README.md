# News Tower Wiki
This is the github repository for the News Tower Wiki. This wiki is a fan made wiki created with support from the developer Sparrow Night and publisher Twin Sails Interactive.

https://newstower.wiki.gg

## How it's made
The wiki will be written in plain text using markdown styling, and the wiki style will be done in CSS. For more information, see the style guide.

## Joining the team
If you wish to join the editing team and contribute to the News Tower Wiki, please contact me on the News Tower Discord #wiki channel.

## Some helpful links
Before diving straight into the wiki, it is helpful to understand how wikis are are organized and structured. https://workingwithmediawiki.com/book/ is a great place to start.
